# 人才盘点系统 Demo

一个前后端一体化（前端页面 + 本地浏览器存储）的示例网站，支持：

1. 选择部门
2. 选择员工
3. 按员工职级自动加载工作行为并评分（1-5分）
4. 自动计算工作行为总分（平均分）
5. 选择绩效评分（1-5分）
6. 在后台表格查看员工评分，并自动映射到九宫格（横轴绩效、纵轴行为总分）

## 运行方式

直接用浏览器打开 `index.html`，或使用本地静态服务：

```bash
python3 -m http.server 8000
```

然后访问：`http://localhost:8000`

## 说明

- 示例员工包含：技术部、市场部、人力资源部。
- 评分记录保存于浏览器 `localStorage`（键名：`talent-review-records`）。
