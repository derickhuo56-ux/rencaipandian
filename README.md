# 人才盘点系统（联网版）

本项目提供两个分离界面：

- **用户评分端**：`/user.html`
  - 选择部门
  - 选择员工
  - 系统后台自动按“员工 → 职级 → 行为”关联评分项（用户界面不展示关联过程）
  - 行为评分（1-5）自动计算行为总分（平均分）
  - 选择绩效评分（1-5）并提交
- **后台管理端**：`/admin.html`（需密码登录）
  - 查看所有员工评分记录
  - 九宫格分布（横轴绩效评分，纵轴行为总分）
  - 修改部门/员工/职级/行为配置
  - 下载 Excel 模板并批量上传更新配置

## 运行

```bash
python3 server.py
```

默认地址：`http://localhost:3000`

## 后台登录

- 默认密码：`admin123`
- 可通过环境变量修改：

```bash
ADMIN_PASSWORD=你的密码 python3 server.py
```

## Excel 模板说明

后台可下载模板，包含两个 Sheet：

1. `员工配置`：列名 `部门`、`员工姓名`、`职级`
2. `行为配置`：列名 `职级`、`工作行为`

上传后系统会覆盖当前组织与行为配置。
